# Clean Data - Excel & CSV Cleaner

A powerful web tool to clean messy Excel and CSV files in seconds.

![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)
![Django](https://img.shields.io/badge/Django-092E20?style=for-the-badge&logo=django&logoColor=white)
![Pandas](https://img.shields.io/badge/Pandas-150458?style=for-the-badge&logo=pandas&logoColor=white)

## ✨ Features

- Upload Excel (.xlsx) or CSV files
- Automatically remove duplicate records
- Remove empty rows
- Instant download of the cleaned file
- **No data stored** on the server (Privacy First)

## 🛠 Tech Stack

- **Backend**: Django
- **Data Processing**: Pandas + NumPy
- **Frontend**: Tailwind CSS
- **Deployment**: Gunicorn + Nginx

## 🚀 Quick Start

```bash
git clone <repository-url>
cd clean-data
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
